# Ejercicio 3.1

Escribe un programa simple de dos líneas que solicite al usuario un número entero y que imprima como resultado `False` si dicho número es menor que `100`, y `True` si el número es mayor o igual que `100`.

### Datos de prueba

```
número = 5
resultado = False
******************
número = 99
resultado = False
******************
número = 100
resultado = True
******************
número = 101
resultado = True
******************
número = -5
resultado = False
******************
número = +123
resultado = True
******************
```
