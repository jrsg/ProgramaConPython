# Ejercicio 4.16

Elabora un programa que calcule una contraseña generada aleatoriamente
para cada usuario.

La contraseña debe contar con las siguientes características:
1. 2 letras minúsculas.
2. 2 letras mayúsculas.
3. 2 números.
4. 2 símbolos.

Ejemplo de una ejecución.

```
rsanchez -> a5$X1mB@
...
ymartinez -> U_3wh#9C
```
